概要

SVODプランは、アプリのお客様が特定のビデオセットを購入する方法を定義します。SVOD プランは Brightcove Beacon で作成する必要があります。 SVOD 計画の作成資料。また、SVOD プランの使用において重要なのは、Video Cloud アカウントの動画用に作成された Video Cloud カスタム フィールドです。 Video Cloud カスタム フィールドの作成資料。具体的には、著作権管理フィールド使用されます。

アクセス権管理のカスタムフィールドについて

アクセス権管理のカスタムフィールドは、次の項目で構成されます。

  • beacon.rights.< >カウンタ.type
  • beacon.rights.< >カウンター.startDate
  • beacon.rights.< >カウンター.endDate
  • beacon.rights.< >カウンター.devices
  • beacon.rights.<カウンター >。locationSperMIT
  • beacon.rights.< >カウンター.locationsDeny
  • beacon.rights.< >counter .packageName
  • beacon.rights.< >カウンター.adConfiguration

まず、 <カウンター>それぞれの名前の一部。これは、権利管理フィールドの複数のセットを作成したい場合に含まれています。 <カウンター>が利用されるため、配列型の構文を使用して関連するフィールドをグループ化できます。

たとえば、休日前後にSVODプランを作成し、次のようなパッケージを用意したいとします。

  • 地球の日
  • アーバー・デイ
  • 新年

Video Cloud に移動して、次のように 0 ～ 2（または 1 ～ 3）のカウンタ値を使用して、3 つのアクセス権管理フィールドを作成する必要があります。

要約すると、使用するパッケージごとに、アクセス権管理フィールドのカウンタセットを作成する必要があります。

パッケージとカスタムフィールドの同様の権限

データを割り当てて、取り込んだときに動画に適用したカスタムフィールドは、Brightcove Beacon でパッケージを作成するときのフォームフィールドと同じように見えます。たとえば、カスタム フィールドbeacon.rights.<カウンター>.locationsDenyそしてパッケージの拒否された場所フォーム フィールドが競合しているように見える場合があります。これらの競合を考えるとき、2つの異なる質問があります。

  • 動画の権利で再生が許可されていますか？
  • ユーザーはパッケージの購読を許可されていますか？

次のシナリオについて考えてみましょう。

  • の終了日beacon.rights.<カウンター>.endDateカスタム フィールドは、パッケージ自体の終了日より後です。つまり、動画の権利により、カスタムフィールドの終了日まで視聴できますが、ユーザーがパッケージにサブスクライブできなかったため、たとえば動画の視聴コストが下がりました。
  • 米国、カナダ、メキシコの許可された場所を持つパッケージが作成され、そのパッケージがカスタム フィールドのビデオに割り当てられます。 beacon.rights.<カウンター>.packageName .また、ビデオのカスタム フィールドでは、UK はbeacon.rights.<カウンター>.locationsPermitカスタムフィールド。パッケージはビデオに割り当てられていますが、英国では購入できません。このビデオは英国で再生できますが、英国のカスタマー向けのパッケージの一部にはできません。
  • 米国、カナダ、メキシコの許可された場所を持つパッケージが作成されます。カスタムフィールドでは、米国とカナダがビデオに追加されていますbeacon.rights.<カウンター>.locationsDenyカスタムフィールド。ここでは、米国、カナダ、メキシコのお客様はパッケージを購読できますが、米国とカナダが拒否されている特定の動画は、ビデオがパッケージに含まれていても、それらの場所のお客様がビデオを表示できなかったことを意味します。

パッケージへの単一のビデオの割り当て

1 つのビデオをパッケージに割り当てるには、Video Cloud Studio を使用する必要があります。以下の手順を実行します。

  1. Studio で、メディアモジュール。
  2. パッケージに追加している動画のタイトルをクリックすると、その動画のプロパティが表示されます。
  3. が表示されるまで下にスクロールします。カスタムフィールドセクションをクリックし、編集ボタン。
  4. Brightcove Beacon で作成された SVOD プランの名前をbeacon.rights.<カウンター>.packageName分野。
  5. パッケージを使用しているため、設定する必要がありますbeacon.rights.<カウンター>.typeにSVOD .
  6. カスタム フィールドのリストの一番下までスクロールし、保存ボタン。

複数のビデオをパッケージに割り当てる

複数のビデオをパッケージに割り当てるには、Video Cloud Studio を使用する必要があります。次のスクリーンショットは、手順を実行するときに役立ちます。

  1. Studio で、メディアモジュールを開き、パッケージに追加する各ビデオの前にあるチェックボックスをオンにします。
  2. クリッククイック編集ボタン。
  3. のクイック編集セクションが画面の右側に表示され、Brightcove Beacon で作成された SVOD プランの名前をbeacon.rights.<カウンター>.packageName分野。
  4. パッケージを使用しているため、設定する必要がありますbeacon.rights.<カウンター>.typeにSVOD .
  5. の下部にクイック編集セクションで、保存ボタン。
