概要

以下のプラットフォームで顧客からの支払いを許可するには、Stripeアカウントが必要です。

  • Web
  • LGスマートテレビ
  • サムスンのスマートテレビ
  • ソニースマートTV
  • 東芝スマートテレビ

これは、Web アプリとすべてのスマート TV アプリに Stripe が必要であることを意味します。以外Android TV と Fire TV。

次の手順では、Brightcoveビーコンで Stripe を使用するプロセスの概要を説明します。

  1. 基本情報（メールアドレス、名前、パスワード）を入力してアカウントを作成します。
  2. ビジネスに関する情報を提供して、アカウントを有効にしてください。
  3. ストライプ API キーを取得します。
  4. Brightcoveの従業員に API キーを提供して、Brightcoveのビーコンエクスペリエンスに追加できるようにします。

Stripeアカウントの作成とアクティブ化

  1. 参照してくださいストライプのホームページ.
  2. クリック今すぐ開始ボタン。
  3. 基本情報を入力してアカウントを作成します。
  4. 指示に従って電子メールアドレスを確認します。
  5. 上で支払いを有効にするダイアログ、クリック今のところスキップ.
  6. 上で家画面の右側に、コピーできる場所が表示されます発行可能なキーと秘密鍵.
  7. 実際のキーを取得するには、個人情報とビジネス情報を入力する必要があります。これを行うには、ツール ヒントのリンク内のリンクをクリックするか、有効化を続けるの左上近くにあるボタン家ページ。
  8. 必要事項の入力が完了したら、ページに戻ります家ページを開いて、ライブ キーを取得します。

ストライプをBrightcoveビーコンに接続する

Stripe API キーを取得したら、Brightcoveのビーコンエクスペリエンスを設定しているBrightcoveの担当者にそのキーを渡します。Stripe API キーを Brightcove ビーコンエクスペリエンスに関連付けて、Stripe をアプリで使用するように設定すると、お客様のアカウントに関連付けられます。
