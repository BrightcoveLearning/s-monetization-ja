Steps to Create a new Coupon

Note: Alternatively you can create the coupons directly in Stripe.

To create new coupons/vouchers in Brightcove Beacon, follow these steps:

  1. In Beacon Classic click the Commerce tab.
  2. In the left menu, select Coupons by Stripe.
  3. Click the Add new Voucher button.
  4. In the Id field, type the code that users will use.
  5. For the Name field, use a User-friendly name that can be displayed in invoices.
  6. For the Duration field, enter the duration of the coupon in months.
  7. In the Discount type you have the option to specify the discount in currency or percentage. For example, if the discount is 3 months free, set the duration to Repeating with 3 months and Discount Type as 100%.
  8. In the Limitations section, you can limit the number of redemptions or date to use the coupon. For example, assume that the coupon can only be used until the end of August, or that only the first 100 people will get the coupon.
  9. In the Packages/Subscription Associated section, specify the subscriptions that can be used with the coupon. If you do not select a subscription, then it will apply to all package subscriptions.
