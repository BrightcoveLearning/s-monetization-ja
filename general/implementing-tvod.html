Introduction

Transactional Video On Demand (TVOD) is a monetization package where viewers can pay a one-time fee to either purchase or rent an OTT asset. A TVOD package can be associated with one asset or a bundle of assets, like a series or season. The viewer can be alloted a certain amount of time to view the asset, or it can be unlimited.

Unlike SVOD where viewers pay a subscription to access video content, TVOD viewers pay a fee for each piece of content they want to view.

TVOD packages give the viewers the following options for watching content:

  • Rent
  • Buy

Creating a TVOD package

To create a TVOD package in Brightcove Beacon, follow these steps:

  1. Login to Brightcove Beacon.

    1. In the top navigation, select Commerce
    2. In the left navigation, select TVOD Plans
    3. click Add New TVOD Plan

  2. For the Package tab, enter details for your TVOD Plan.

    This table contains the fields you will find in the Package tab:

    Field Required Description
    Name Yes Name of the TVOD package
    TVOD Plan Type Yes Select one:
    • Multiple Assets - Associate a bundle of assets with this plan; for example, viewers can purchase/rent a full series or season
    • Individual Assets - Viewers can buy or rent 1 asset
    Purchase Type Yes Select one:
    • Buy - Allow viewers to purchase an asset or bundle of assets
    • Rent - Allow viewers to rent an asset or bundle of assets for a period of time
    Rent period No Number of days an asset is available for viewing once it has been rented. This field appears when the Purchase Type is set to Rent.
    Status Yes Select one of the following types:
    • Published
    • Unpublished
    Allow Downloads Yes Set to Yes to allow this series to be downloaded by the user for offline playback. For details, see the Offline Playback with Brightcove Beacon Apps document.
    Payment methods to use Yes Select one of the following types:
    • Stripe
    • Beacon Store

  3. If you are using Stripe, select Stripe to see Payment methods details.

    Availability is for the TVOD package. For example, if the Permitted locations is set for the US, then all viewers will be able play a movie or asset associated with this package, but only viewers in the US can buy/rent the asset.

    This table contains the fields Stripe payment details:

    Field Required Description
    Start Time Yes Schedule a starting date this package will be available
    End Time Yes Schedule an ending date when this package will no longer be available
    Devices to publish Yes Add the devices you wish to publish to. Select the Select / Unselect All Devices to see a list of available devices.
    Permitted locations Yes

    Select the locations where you want to stream your content, if no restrictions enter World.

    If you don't specify at least 1 permitted location, then you may see this message from your app:
    Denied locations No Select the locations where you DO NOT want to stream your content. Enter World if you have no restrictions.
    Price Yes Specify the price that users must pay to view content associated with this package.
    Currency Yes Specify the currency type for the package price.
    Product Description Yes Product Description is sent to Stripe so that you can identify the package.

  4. If you are using the Beacon store, select Beacon Store to see Payment methods details.

    This table contains the fields Beacon Store payment details:

    Field Required Description
    Beacon Store ID Yes TVOD with the Beacon Store requires setup of the SKU. The Beacon Store ID is the package_name sent to the Beacon Store using the API.

  5. For the Textual Data tab, enter details for your plan.

    This table contains the fields you will find in the Textual Data tab:

    Field Required Description
    Pack Name Yes Display name shown to viewers
    Headline Yes Description shown to viewers, for example Holiday Special!

  6. For the Non-Textual Data tab, enter details for your plan.

    This table contains the fields you will find in the Non-Textual Data tab:

    Field Required Description
    Package Image No Supply an image for use next to the package name
  7. Click Create Package.

Assigning TVOD packages to content

To add content to TVOD packages, edit each asset and associate it with applicable packages. The following content types can be used with TVOD packages:

  • Movies
  • Series, Seasons, and Episodes
  • Channels and Live Events

These steps walk through adding a TVOD package to a movie, but the process is similar for other asset types.

  1. Return to Brightcove Beacon. In the top navigation, select Movies. Click the Name link for your movie.

  2. Click the Rights & Scheduling tab.

  3. In the Availability Details section, click Rights Type and select Buy/Rent - TVOD.

  4. Click in the Select Package input area and select one or more TVOD packages that you want associated with this movie.

  5. Click Update Movie.

  6. Once you associate TVOD packages with content, then you will see that content listed in the package details.

    Return to your TVOD plan.

    1. In the top navigation, select Commerce
    2. In the left navigation, select TVOD Plans
    3. click Content tab

    Here, you will see content associated with this plan.
