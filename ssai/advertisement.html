Overview

To create an advertisement configuration you will need to do the following:

  • Obtain an ad tag server URL from your ad service provider. This will be used in Video Cloud Studio to create your SSAI ad configuration.
  • In Video Cloud Studio, create your SSAI ad configuration. At completion of the task you will copy the ad server ID to create your advertisement configuration in Beacon Classic.
  • In Beacon Classic create your advertisement configuration, which you will then be able to apply to your video content.

Create an SSAI ad configuration ID

Follow these steps to create an SSAI ad configuration:

  1. In {% if site.product_short == "Beacon" or site.product_short == "CorpTV" %} Studio {% elsif site.product_short == "Beacon Classic" %} Video Cloud Studio {% endif %}, navigate to Admin â†’ Server-Side Ad Settings.
  2. From the Create Ad Configuration dropdown, select VOD
  3. Add a name for the ad config.
  4. Select your ad response - VMAP or VAST is recommended. If you use VAST, you will need to add cuepoints to the videos to specify where ad breaks should appear.
  5. Paste in your ad tag.
  6. Click Save.
  7. Find your new ad configuration in the table and copy your ad config ID. You will need this value for the Beacon advertising configuration.

Create an advertisement configuration

To create an advertisement configuration, follow these steps:

  1. In Beacon Classic, in the top navigation select Advertisement .
  2. Click Add New Configuration.
  3. Name the Ad Configuration.
  4. For the Provider select Brightcove SSAI
  5. Paste in the SSAI ad config ID copied in the steps just above into VOD Ad Config ID.
  6. Click Create New Advertisement.
