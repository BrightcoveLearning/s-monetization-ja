概要

アドバタイズメント設定を作成するには、次の手順を実行する必要があります。

  • 広告サービスプロバイダから広告タグサーバーのURLを取得します。これは、VideoCloudStudioでSSAI広告設定を作成するために使用されます。
  • VideoCloudStudioでSSAI広告設定を作成します。タスクが完了すると、広告サーバーIDをコピーしてBeaconClassicで広告設定を作成します。
  • BeaconClassicでアドバタイズメント設定を作成し、ビデオコンテンツに適用できます。

SSAI広告設定IDの作成

SSAIアドコンフィギュレーションを作成する手順は、次のとおりです。

  1. {% if site.product_short == "Beacon" or site.product_short == "CorpTV" %} Studio {% elsif site.product_short == "Beacon Classic" %} Video Cloud Studio {% endif %} に移動します。管理者 → サーバー側の広告設定.
  2. から広告構成の作成ドロップダウン、選択VOD
  3. 広告設定の名前を追加します。
  4. 広告のレスポンスを選択してください-VMAPまたはVASTを推奨します。VASTを使用する場合は、ビデオにキューポイントを追加して広告ブレークを表示する場所を指定する必要があります。
  5. 広告タグを貼り付けます。
  6. クリック保存.
  7. テーブルで新しい広告設定を検索し、広告設定IDをコピーします。ビーコンアドバタイジングの設定には、この値が必要です。

アドバタイズメント設定の作成

  1. Beacon Classic では、上部のナビゲーションで選択します広告.
  2. クリック新しい構成を追加.
  3. 広告の設定に名前を付けます。
  4. のためにプロバイダ選択するブライトコーブ SSAI
