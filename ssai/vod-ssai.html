Overview

Server-side ads are embedded in the video stream so they cannot be blocked by ad blockers in the browser. It works with both DRM-protected and non-protected content. Brightcove Beacon supports VOD with SSAI.

Here are the steps for adding server-side ads:

  1. In {% if site.product_short == "Beacon" or site.product_short == "CorpTV" %} Studio {% elsif site.product_short == "Beacon Classic" %} Video Cloud Studio {% endif %}, create an ad configuration.
  2. In {% if site.product_short == "Beacon" or site.product_short == "CorpTV" %} Studio {% elsif site.product_short == "Beacon Classic" %} Video Cloud Studio {% endif %}, set the custom field values for your video.
  3. In {% if site.product_short == "Beacon" or site.product_short == "CorpTV" %} Studio {% elsif site.product_short == "Beacon Classic" %} Video Cloud Studio {% endif %}, add your video to a playlist.
  4. In {% if site.product_short == "Beacon" or site.product_short == "CorpTV" %} Studio {% elsif site.product_short == "Beacon Classic" %} Video Cloud Studio {% endif %}, import your videos and playlists, then set advertising options.

Best practices

Brightcove recommends following these best practices when adding server-side ads to VOD content:

  • Create your ad configuration in Video Cloud Studio.
  • Assign the ad configuration to videos using the custom fields in Video Cloud Studio.
  • Only use the Advertisement section in Brightcove Beacon for live events and channels.

Creating an ad configuration

Here you will create an SSAI ad configuration to define the ads displayed in your Brightcove Beacon VOD asset.

  1. Log in to {% if site.product_short == "Beacon" or site.product_short == "CorpTV" %} Studio {% elsif site.product_short == "Beacon Classic" %} Video Cloud Studio {% endif %}.
  2. In the primary navigation, click Admin and select Server-Side Ad Settings. A list of all ad configurations will be displayed.
  3. To create a new ad configuration, click Create Ad Configuration.
  4. From the dropdown, select VOD as the type.
  5. Check to be sure you are seeing the Server-Side Ad Settings form.
  6. Give your ad configuration a Name.
  7. Select the Ad Server Response:
    Ad Server Response Details
    VAST 3.0 Digital Video Ad Serving Template
    DFP VMAP Video Multiple Ad Playlist
    DFP (Ad Rules) Proprietary format for DFP/Google for Publishers
    Smart XML Proprietary format for FreeWheel

    Please check with your ad operations manager for the correct value to select.
  8. Enter the Ad Tag (server URL).
  9. Click Save.
  10. Copy the ID of your Ad Configuration, as you will need it later to configure your VOD asset.

Setting custom field values

Note: This section uses Video Cloud rights management custom fields. See the Custom Fields for Monetization document for information on creating and using rights management custom fields.

In this section, you will learn how to configure your VOD asset with the necessary settings to display SSAI in your Brightcove Beacon apps.

  1. In the primary navigation, click Media.
  2. Click the video to which you want to add SSAI.
  3. Scroll down to the Custom Fields section.
  4. Click Edit.
  5. Scroll down and locate the beacon.rights.0.adConfiguration field, then paste the ID of your previously created Ad Configuration.
  6. Keep scrolling down and in the beacon.rights.0.type field write AVOD to categorize this VOD as and Advertising VOD.
  7. Click Save.
    By default, you can add up to 3 SSAI ID and type configurations in the Custom Fields section.
  8. In the top menu, click Back to Videos.

Adding videos to a Playlist

In this section, you will add your VOD asset to a playlist.

  1. In {% if site.product_short == "Beacon" or site.product_short == "CorpTV" %} Studio {% elsif site.product_short == "Beacon Classic" %} Video Cloud Studio {% endif %}, click Media.
  2. Click the checkbox for your video.
  3. Click More.
  4. Click Add to Playlist.
  5. Select the playlist to which you want to add your video.
  6. Click Add.
  7. Once your VOD asset has been added to the playlist, go to the Brightcove Beacon platform.

Importing videos and playlists

In Brightcove Beacon, you can ingest/import videos and playlists from Video Cloud.

  1. Log in to your Beacon Classic.
  2. In the top menu, select Tools, which is the wrench icon.
  3. In the left navigation, select Ingestion.
  4. Click Update Brightcove Videos and/or Update Brightcove Playlists.
  5. Your videos and playlists from Video Cloud should now be in Brightcove Beacon. In the top menu, click the Playlists tab.
  6. Select your playlist to see the videos it contains.
  7. Check to be sure you can see all the VOD assets associated with that playlist in the left table of contents.
  8. In the Brightcove Beacon top menu, click in the Movies tab.
  9. Click on your VOD asset.
  10. Click the Rights & Scheduling tab.
  11. In the Rights Type menu, you should see Advertising - AVOD selected.
  12. In the Advertisement Configuration menu, you should see your previously created ad configuration id.
  13. To see your videos in your app right away, you need to clear the cach. In the top menu, click the Tools (wrench) button, then clear the cache.
  14. In the left navigation, select Cache.
  15. Select the Cache Purge button.
  16. Open your Brightcove Beacon App and click in the Page tab on which your playlist was added. (All the event names and pictures on the image are for example purposes only).
    To visualize your Playlist in the Brightcove Beacon App, you must have added your playlist to a Brightcove Beacon Page first. To know more about how to add a playlist to a Page please visit the following documentation.
  17. Click the movie that you have configured with server-side ads.
  18. Click the Play button and you should see your video play with ads.
