概要

サーバーサイド広告は動画ストリームに埋め込まれているため、ブラウザーの広告ブロッカーによってブロックすることはできません。DRM で保護されたコンテンツと非保護コンテンツの両方で動作します。Brightcoveビーコンは、SSAIでVODをサポートしています。

サーバーサイド広告を追加する手順は次のとおりです。

  1. In {% if site.product_short == "Beacon" or site.product_short == "CorpTV" %} Studio {% elsif site.product_short == "Beacon Classic" %} Video Cloud Studio {% endif %},広告構成を作成します。
  2. In {% if site.product_short == "Beacon" or site.product_short == "CorpTV" %} Studio {% elsif site.product_short == "Beacon Classic" %} Video Cloud Studio {% endif %},カスタム フィールドの値を設定するあなたのビデオのために。
  3. In {% if site.product_short == "Beacon" or site.product_short == "CorpTV" %} Studio {% elsif site.product_short == "Beacon Classic" %} Video Cloud Studio {% endif %},動画をプレイリストに追加する.
  4. In {% if site.product_short == "Beacon" or site.product_short == "CorpTV" %} Studio {% elsif site.product_short == "Beacon Classic" %} Video Cloud Studio {% endif %},動画とプレイリストをインポートする、広告オプションを設定します。

ベストプラクティス

Brightcoveでは、VOD コンテンツにサーバーサイド広告を追加する場合は、次のベストプラクティスに従うことをお勧めします。

  • VideoCloudStudioで広告設定を作成します。
  • VideoCloudStudioのカスタムフィールドを使用して、ビデオに広告設定を割り当てます。

カスタムフィールドの値の設定

注:このセクションでは、VideoCloud権限管理カスタムフィールドを使用します。を参照してください収益化のためのカスタム フィールドRights Management カスタム フィールドの作成と使用については、ドキュメントを参照してください。

このセクションでは、Brightcove BeaconアプリケーションでSSAIを表示するために必要な設定でVODアセットを設定する方法について説明します。

  1. プライマリ ナビゲーションで、メディア。
  2. SSAI を追加するビデオをクリックします。
  3. 下にスクロールしてカスタムフィールドセクション。
  4. クリック編集。
  5. 下にスクロールして、 beacon.rights.0.adConfigurationフィールドに、以前に作成した Ad Configuration の ID を貼り付けます。
  6. 下にスクロールし続け、 beacon.rights.0.typeフィールド書き込みアヴォドこの VOD を広告 VOD として分類します。
  7. クリック保存。
    デフォルトでは、最大 3 つの SSAI ID とタイプの設定をカスタムフィールドセクション。
  8. 上部のメニューで、動画に戻る.

動画とプレイリストのインポート

Brightcove Beacon では、Video Cloud から動画や再生リストを取り込む/読み込むことができます。

  1. ビーコンクラシックにログインします。
  2. 上部のメニューで、ツール、レンチ アイコンです。
  3. 左側のナビゲーションで、摂取.
  4. クリックブライトコーブの動画を更新するおよび/またはBrightcove プレイリストを更新します。
  5. これで、Video Cloud の動画と再生リストが Brightcove Beacon に表示されます。上部のメニューで、プレイリストタブ。
  6. プレイリストを選択すると、そのプレイリストに含まれる動画が表示されます。
  7. 左側のコンテンツウィンドウに、そのプレイリストに関連付けられているすべての VOD アセットが表示されていることを確認します。
  8. Brightcove ビーコンのトップ メニューで、映画タブ。
  9. VOD アセットをクリックします。
  10. クリック権利とスケジュールタブ。
  11. の中に権利タイプメニュー、あなたが見るはずです広告 - AVOD選択されました。
  12. の中に広告構成メニューに、以前に作成した広告設定 ID が表示されます。
  13. アプリですぐに動画を表示するには、キャッシュをクリアする必要があります。上部のメニューで、ツール(レンチ) ボタンをクリックしてから、キャッシュをクリアします。
  14. 左側のナビゲーションで、キャッシュ.
  15. を選択キャッシュパージボタン。
  16. Brightcove ビーコン アプリを開き、ページプレイリストが追加されたタブ。（画像上のすべてのイベント名と画像は、例のためだけのものです）。
