このライブ広告構成プロセスは、ライブ チャネルが SSAI で有効化されている場合にのみ必要です。そうでない場合は、これらの手順は必要ありません。

前提条件

  • VideoCloudでSSAIを有効にしたライブチャネルを作成します。
  • ビデオクラウドで広告タグを作成して設定します。
  • Brightcove Beaconでチャネルを作成します。
  • Brightcove Beaconでアドバタイズメント設定を作成します。
  • 権限の設定 & チャネルでのスケジューリング。

ビデオクラウド広告の設定

  1. Video Cloud Studio で、住むモジュール。
  2. Brightcove Beaconで使用するLiveSSAIチャネルを作成または開きます。
  3. の中に制御室、 クリック公開して埋め込む左パネルから。
  4. の中にライブ プレーヤーのオプションセクションで、ドロップダウンから広告設定設定したい。
  5. 新しいものを選択したら標準埋め込みコード現れる。
  6. を抽出しますadConfigIdイメージに示すように、adConfigId に割り当てられている URL の一部をコピーします。

Brightcoveビーコンでのアドバタイズメントの作成

  1. Brightcove Beaconプラットフォームに戻ります。
  2. クリック広告.
  3. 設定する広告設定を作成または選択します。
  4. 新しい広告の場合は、名前.
  5. 必ずプロバイダに設定されていますブライトコーブ SSAI .
  6. の中にライブ広告構成 IDフィールドに貼り付けますadConfigIdこのドキュメントの前のセクションでコピーした URL。
    ビーコン チャネルを使用する場合、ライブ広告設定 ID value は、広告構成 ID ではなく、ライブ再生トークンです。
  7. クリック新しい広告を作成する.

権限の設定 & Brightcoveビーコンでのスケジューリング

  1. 上部のナビゲーションで、チャネル.
  2. SSAIを設定するチャネルを選択します。
  3. クリック権利とスケジュールタブ。
  4. の権利タイプ、 選択する広告 - AVOD .
  5. の広告構成で、ドロップダウンから、作成した広告構成を選択します。
  6. クリック新しいチャンネルを作成作業を保存します。
